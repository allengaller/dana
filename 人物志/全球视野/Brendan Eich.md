# Brendan Eich：JavaScript 之父，在 10 天内创造了一门统治互联网的语言

Brendan Eich 在 1995 年为 Netscape 浏览器，在 10 天内设计并实现了 JavaScript（最初叫 Mocha）。

这门原本「为了让网页有简单交互」的脚本语言，最终成了互联网上最广泛使用的编程语言，深刻改变了前端、后端（Node.js）、甚至移动和桌面开发。

他的故事是**在巨大约束和极短时间内做出「足够好」决策**的经典案例。

---

## 核心可学习点

### 1. 在极度受限的条件下做出「够用且能进化」的决策
10 天时间、必须兼容 Java 和 Scheme 的要求、浏览器环境限制……他做出了当时最优的权衡。

### 2. 「先跑起来，再变好」有时比「一开始就完美」更重要
JavaScript 最初有很多「糟糕」的设计，但因为它「能用」而赢得了整个互联网。

### 3. 语言的成功往往取决于生态而非语法本身
JavaScript 的真正胜利，来自 DOM、浏览器普及、以及后来无数人的贡献。

### 4. 即使是「快速原型」，也要有长远的品味
Brendan 当时就考虑了函数式特性（first-class functions），这为后来的成功埋下了伏笔。
