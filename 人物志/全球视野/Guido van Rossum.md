# Guido van Rossum：Python 之父，用「可读性」重新定义编程语言

> 「代码的可读性非常重要。代码会被阅读的次数远多于被编写的次数。」  
> —— Guido van Rossum

Guido van Rossum 是 Python 语言的创造者，曾长期担任 Python 的「终身仁慈独裁者」（BDFL）。

Python 以极致的可读性、简洁的语法、以及「人生苦短，我用 Python」的哲学，成了全球最受欢迎的编程语言之一，尤其在数据科学、AI、自动化领域统治地位明显。

Guido 的路径体现了**语言设计中的人文关怀 + 长期社区治理 + 实用主义**。

---

## 核心可学习点

### 1. 把「让普通人也能写出好代码」当作语言设计的最高目标
Python 的设计哲学（PEP 20: The Zen of Python）至今仍是很多语言设计的标杆。

### 2. 「仁慈独裁者」模式的社区治理
在项目早期，由一个人做最终决策，可以保持一致性和快速推进。但也需要知道何时放权。

### 3. 长期主义 + 向后兼容的极致重视
Python 3 的缓慢迁移过程，体现了 Guido 对「不要破坏已有代码」的极度谨慎。

### 4. 语言设计是「品味」的体现
Guido 经常强调「Pythonic」，这其实是一种可被感知的审美和哲学。
