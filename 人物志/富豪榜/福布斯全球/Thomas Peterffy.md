# Thomas Peterffy：从匈牙利难民到电子交易之父的 60 年金融革命

> **"I grew up in a socialist country. There is no hope, no freedom. I came to America because I wanted to build things."**
> — Thomas Peterffy

Thomas Peterffy 是**金融交易领域最重要的技术创新者之一**，却鲜为人知。

1965 年，21 岁的他从匈牙利共产主义政权下移民美国，身无分文，英语不通。他靠自学编程成为华尔街最早用计算机做交易的先驱——1983 年把便携式电脑带上交易大厅（当时被交易所禁止），1993 年创办 Interactive Brokers，用技术把交易成本压到行业最低。

2025 年，Interactive Brokers 管理资产超过 5000 亿美元，Peterffy 个人财富达到 800 亿美元。他证明了：**技术可以摧毁旧制度、重建新秩序**。

## 二、发家史（关键转折）

### 转折一：1965 年逃离匈牙利——"从共产主义到资本主义"

1944 年，Peterffy 出生在布达佩斯一个犹太家庭（二战期间和战后的极端困难时期）。

**背景**：
- 匈牙利在苏联影响下实行计划经济
- Peterffy 年轻时目睹了物资短缺、缺乏自由的现实
- 1965 年，21 岁的他设法移民美国（具体方式未公开，据称是通过奥地利边境）

**到美国后**：
- 身无分文，英语不通
- 第一份工作：在纽约 Queens 做公路测量员
- 自学英语（通过日常沉浸）
- 自学编程（通过可编程计算器）

**关键信号**：**Peterffy 的反市场化信念来自亲身经历**——他亲眼见过计划经济的失败，这成为他终身倡导自由市场的根源。

### 转折二：1969-1977 从程序员到交易员——"发现市场的效率漏洞"

Peterffy 在一家金属交易公司找到编程工作，开始研究金融市场的定价逻辑。

**关键发现**：
- 1970s 的华尔街交易完全靠人工（喊价、手势）
- 期权定价可以用数学模型精确计算（Black-Scholes 公式 1973 年发表）
- **人工交易 = 低效率 = 套利机会**

**1977 年**：Peterffy 用积蓄购买了一个纽约证券交易所（AMEX）的交易席位，开始做期权做市商。

**关键信号**：**Peterffy 是最早意识到"计算机可以打败人工交易员"的人之一**。

### 转折三：1983 年把电脑带上交易大厅——"电子交易的开端"

1983 年，Peterffy 做了一件前所未有的事：**把一台便携式计算机带上了 AMEX 交易大厅**。

**背景**：
- 当时的交易大厅只允许纸笔交易
- Peterffy 的计算机可以实时计算期权价格，比人工快几秒（在交易中，几秒 = 巨大利润）
- 交易所起初允许（因为设备没有明确被禁止），后来试图禁止

**Peterffy 的应对**：
- 当交易所要求"不能用计算机"时，他设计了一个**机械打字装置**——计算机计算价格，机械臂在交易单上"人工"打字
- 这迫使交易所不得不面对一个问题：**什么是"人工"交易？**

**关键信号**：**Peterffy 是"电子交易之父"**——他用技术手段一次次挑战交易所的规则边界，最终推动了整个行业的电子化。

### 转折四：1993 年创办 Interactive Brokers——"让散户也能像做市商一样交易"

1993 年，Peterffy 将他的自营交易公司拆分出面向客户的服务——**Interactive Brokers（IBKR）**。

**核心创新**：
- **超低佣金**：IBKR 的交易费用远低于传统券商
- **全球市场接入**：一个账户可以交易全球 150+ 个市场
- **算法交易工具**：提供 SmartRouting（自动选择最优执行路径）
- **保证金利率低**：IBKR 的保证金利率是行业最低之一

**2007 年**：Interactive Brokers 在纳斯达克 IPO。

**IBKR 的增长**：
- 2007 年 IPO 时：约 50 万客户
- 2025 年：超过 300 万客户
- 管理资产：超过 5000 亿美元
- 日均交易：超过 300 万笔

**关键信号**：**Peterffy 把"做市商的工具"卖给了散户**——这是一种"技术民主化"。

## 三、商业模式与护城河

- **核心业务**：在线券商（Interactive Brokers）
- **护城河类型**：
  - **技术壁垒**（自动化执行系统、全球市场接入）
  - **规模效应**（300 万客户，低边际成本）
  - **价格优势**（行业最低佣金和保证金利率）
- **收入结构**：净利息收入 60% + 佣金/交易费 30% + 其他 10%
- **关键杠杆**：**代码 + 自动化 + 规模**

### "技术驱动的低成本" 模型

```
自动化执行系统 ──→ 低运营成本 ──→ 低佣金吸引客户 ──→ 更多客户
     ↑                                                     ↓
  技术持续改进 ←── 利润再投资 ←── 规模效应 ←── 管理资产增长 ←┘
```

## 六、社会影响力

### 慈善与公益
- **低调**：Peterffy 不像 Gates 或 Bloomberg 那样高调做慈善
- 支持自由市场组织
- 2023 年向佛罗里达州共和党政治候选人捐款

### 政治与公共参与
- **2012 年超级碗广告**：Peterffy 自费在超级碗期间播放了一则**反对政府监管**的广告（30 秒，花费约 400 万美元）
- **政治捐款**：偏向共和党 / 自由意志主义
- **反对央行干预**：公开批评美联储的量化宽松政策
- **支持贸易保护**：近年支持对进口商品征收关税以保护美国制造业（与他的自由市场信念看似矛盾）

### 争议事件
- **IBKR 的"GameStop"事件（2021）**：在 GameStop 散户交易狂潮中，IBKR 因保证金要求提高而被散户指责"限制交易"
- **反监管立场**：Peterffy 的超级碗广告被批评为"亿万富翁反对监管"
- **SEC 调查**：IBKR 历史上多次被 SEC 调查合规问题

## 七、可复用的心法与决策模式

### 心法 1：**"技术可以摧毁旧制度"**

> Peterffy 一生都在用技术挑战金融行业的"人工壁垒"——从把电脑带上交易大厅，到用算法替代人工做市。

**应用**：1983 年便携式电脑、1993 年 IBKR 自动化交易、2000s SmartRouting。

### 心法 2：**"自学是最强的武器"**

> Peterffy 从未上过大学。他靠自学英语、自学编程、自学金融，成为三个领域的专家。

**应用**：从匈牙利测量员到华尔街交易员到科技公司创始人。

### 心法 3：**"亲身经历塑造信念"**

> "I grew up in a socialist country... There is no hope, no freedom."

**应用**：Peterffy 对自由市场的信仰来自匈牙利计划经济的亲身经历。这让他 50 年不动摇。

### 心法 4：**"把专业工具民主化"**

**应用**：IBKR 把原本只有做市商才能使用的交易工具（低佣金、全球市场、算法执行）提供给了散户。
