# 案例研究

本目录收录基于「五层研究方法论」的完整实战案例复盘。

与 `人物志/` 不同，本目录侧重于**匿名或半匿名的真实成长案例**，
重点展示可复制的决策过程、行动路径与结果复盘。

## 内容标准

每篇案例须包含：

- **背景**：人物阶段、领域、约束条件
- **关键决策**：当时做了什么选择，为什么
- **行动过程**：具体做了什么，持续多久
- **结果与数据**：可验证的产出（不要求精确数字）
- **可复制点**：其他人能学到什么
