## 如何使用这个知识库

1. **先读索引**：从 `索引/` 或 AI时代大拿索引 开始，建立全局图景。
2. **精读 2-3 位**：挑最打动你的，逐条抄“今天可以开始的行动”。
3. **内化心法**：把心法抄到自己的卡片/第二大脑，每周复盘实践了哪一条。
4. **对标路径**：找到自己当前阶段（尤其是 3-7 年突破），制定小目标。
5. **持续输出**：每读一篇，写一段「我能立刻开始的 1 件事」并公开分享。

**最佳体验**：用 **Obsidian** 打开整个仓库（支持 wikilinks、Canvas 可视化、图谱）。

也可以用 Typora、VS Code、GitHub 在线或任何 Markdown 阅读器。
