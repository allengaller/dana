## 如何使用

1. **先读索引**：从专题索引建立全局认知
2. **精读 2-3 位**：挑最打动你的大拿，逐条抄写"今天可以开始的行动"
3. **内化心法**：把心法记录到自己的第二大脑，每周复盘
4. **对标路径**：找到当前阶段（尤其 3-7 年突破期），制定小目标
5. **持续输出**：每读一篇，写一段"我能立刻开始的 1 件事"并公开分享

**最佳体验**：用 [Obsidian](https://obsidian.md) 打开整个仓库（支持 wikilinks、Canvas 可视化、图谱）。也可用 Typora、VS Code 或 GitHub 在线阅读。
