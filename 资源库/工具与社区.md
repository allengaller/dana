# 工具与社区：技术人长期作战的装备库

## 推荐工具（第二大脑 + 写作 + 开发）

- **Obsidian / Logseq**：构建第二大脑的最佳工具
- **Excalidraw / Draw.io**：画架构图
- **Git + GitHub**：版本控制 + 个人品牌
- **Typora / VS Code + Markdown**：日常写作

## 推荐社区

- **GitHub**：最重要
- **中文技术社区**：掘金、InfoQ、博客园（选择性参与）
- **英文**：Hacker News、Reddit (r/programming, r/cscareerquestions)
- **特定领域**：根据你的方向加入高质量 Slack/Discord/微信群
