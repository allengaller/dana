# 非线性路径

不是每个人都走「大厂 → 专家 → Leader」的线性路线。

**教育家型**（阮一峰、刘大拿等）：用长期输出建立影响力与经济体  
**独立开发者/思想者**（冯大辉等）：用开源 + 个人品牌 + 内容建立独立性  
**极致深度专家**（丁奇等）：单点打到全球顶尖

这些路径对「自我驱动」要求更高，但自由度也更高。
